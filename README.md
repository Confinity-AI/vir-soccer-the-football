# VIR Soccer — The Football

Next.js site for **VIR Soccer Academy**, living under `AmalgamSites/sites/vir_soccer`.

## Scripts

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build
npm start
```
